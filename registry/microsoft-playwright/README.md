# Playwright

微软出品的端到端测试框架。跨浏览器（Chromium/Firefox/WebKit），自动等待，trace viewer。

## 🪨 石板地基

**来源**: [microsoft/playwright](https://github.com/microsoft/playwright)  
**生态**: npm  
**版本**: 1.x

## ✅ 适合做什么

- 端到端测试
- 跨浏览器测试
- CI/CD 集成
- 截图对比
- 网络拦截

## 🔑 关键词

playwright · 测试 · 端到端
