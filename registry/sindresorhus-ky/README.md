# Ky

基于 Fetch API 的轻量 HTTP 客户端。零依赖，支持重试、超时、hook。比 axios 更现代。

## 🪨 石板地基

**来源**: [sindresorhus/ky](https://github.com/sindresorhus/ky)  
**生态**: npm  
**版本**: 1.x

## ✅ 适合做什么

- 现代浏览器/Node
- Fetch API 场景
- 轻量替代 axios
- REST API 调用
- 需要重试
