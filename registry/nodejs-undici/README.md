# Undici

Node.js 官方 HTTP 客户端。速度最快的 Node HTTP 实现，HTTP/1.1 + HTTP/2。Node 核心团队维护。

## 🪨 石板地基

**来源**: [nodejs/undici](https://github.com/nodejs/undici)  
**生态**: npm  
**版本**: 7.x

## ✅ 适合做什么

- Node.js 项目
- 高并发 HTTP
- HTTP/2
- 性能优先
- 替代 node-fetch
