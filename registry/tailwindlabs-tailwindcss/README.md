# Tailwind CSS

实用优先（utility-first）CSS 框架。直接在 HTML 里写样式，JIT 引擎按需生成。现代 Web 开发标准。

## 🪨 石板地基

**来源**: [tailwindlabs/tailwindcss](https://github.com/tailwindlabs/tailwindcss)  
**生态**: npm  
**版本**: 4.x

## ✅ 适合做什么

- 现代 Web 项目
- 快速原型
- 设计系统
- 响应式布局
- React/Vue/Svelte
