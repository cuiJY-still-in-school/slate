# NextAuth.js

Next.js 认证库。支持 OAuth、Credentials、Magic Link 等 80+ provider。内置 session 管理。

## 🪨 石板地基

**来源**: [nextauthjs/next-auth](https://github.com/nextauthjs/next-auth)  
**生态**: npm  
**版本**: 5.x

## ✅ 适合做什么

- Next.js 项目
- OAuth 登录
- 多 provider
- session 管理
- SSR 认证

## 🔑 关键词

认证
