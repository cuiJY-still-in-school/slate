# Valibot

模块化、可 tree-shake 的 schema 校验库。打包体积比 Zod 小 98%，API 相似。追求极致 bundle size 的首选。

## 🪨 石板地基

**来源**: [fabian-hiller/valibot](https://github.com/fabian-hiller/valibot)  
**生态**: npm  
**版本**: 1.x

## ✅ 适合做什么

- 追求小 bundle
- 边缘计算
- Serverless
- Zod 替代
- tree-shaking
