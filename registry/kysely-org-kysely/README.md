# Kysely

端到端类型安全的 SQL 查询构建器。写 SQL，得 TypeScript 类型。比 Prisma 更灵活，比 Drizzle 更接近 SQL。

**来源**: [kysely-org/kysely](https://github.com/kysely-org/kysely)  
**生态**: npm  
**版本**: 0.x

## ✅ 适合做什么

- SQL 专家
- 复杂查询
- 类型安全 SQL
- Prisma/Drizzle 替代
- PostgreSQL
